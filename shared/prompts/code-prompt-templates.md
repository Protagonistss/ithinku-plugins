# 代码相关提示词模板

这个文件包含常用的代码相关提示词模板，可以在不同插件中复用。

## 使用说明

### 模板占位符说明
- `[...]` 部分需要替换为实际内容
- 可根据具体需求调整模板细节
- 建议结合项目特点自定义模板

### 最佳实践
1. 提供足够的上下文和细节
2. 使用列表和分段组织信息
3. 包含具体的代码示例
4. 明确说明期望的输出
