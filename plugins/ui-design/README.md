**拒绝 AI Default Slop。**
本插件致力于帮助开发者突破平庸的 "AI 审美"，创造出令人印象深刻、细节丰富且具有独特个性的用户界面。

## 🎯 核心理念 (Philosophy)

- **Bold Aesthetics**: 每一个设计都应有明确且大胆的风格导向（如 Brutalism, Editorial, Glassmorphism）。
- **Anti-Patterns**: 严禁使用泛滥的紫色渐变、默认圆角和毫无个性的布局。
- **Refined Details**: 强调排版 (Typography)、动效 (Motion) 和深度 (Depth) 的精细打磨。

## 🚀 核心能力

- **Core Design（技能名：ui-design-core）**: 生成具有“设计感”的代码。支持 Tailwind CSS, CSS Modules 等。
- **A11y Check（技能名：ui-design-a11y）**: 确保独特设计的同时不牺牲无障碍访问性 (WCAG)。

## 🎨 快速体验

```bash
# 生成一个野兽派风格的博客首页
/ui-design 生成一个 Neo-Brutalism 风格的博客首页，使用高对比度黑白配色和巨大的衬线标题。

# 拒绝平庸，重构现有设计
/ui-design 将这个普通的登录页重构为 Cyberpunk 风格，添加霓虹光晕和 Glitch 动效。
```

## ⚙️ 配置

```json
{
  "ui-design": {
    "framework": "tailwind",
    "theme": "dark" // 推荐使用 dark 模式以获得更强的视觉冲击力
  }
}
```
