# UI/UX Design Skill

## 技能描述

专业的UI/UX设计技能，帮助创建直观、美观且用户友好的界面设计，遵循现代设计原则和最佳实践。

## 核心功能

- 📊 **用户研究** - 用户画像、用户旅程、需求分析
- 🎨 **界面设计** - 布局、视觉层次、信息架构
- 🔄 **交互设计** - 流程设计、状态反馈、微交互
- 📱 **原型设计** - 线框图、高保真原型、交互原型
- ✅ **可用性测试** - 测试方案、反馈收集、迭代优化

## 快速使用

```bash
# 设计一个登录页面
/frontend-design 设计登录页面的UI和用户体验

# 优化注册流程
/frontend-design 优化用户注册流程，减少流失率

# 创建仪表板布局
/frontend-design 设计数据分析仪表板的界面布局
```

## 配置

```json
{
  "uiUxDesign": {
    "designPrinciples": [
      "consistency", // 一致性
      "feedback",    // 反馈
      "simplicity",  // 简洁性
      "efficiency"   // 效率性
    ],
    "userPersonas": {
      "primary": "主要用户群体描述",
      "secondary": ["次要用户群体1", "次要用户群体2"]
    },
    "designSystem": {
      "spacingUnit": 8,
      "borderRadius": 4,
      "shadows": true
    }
  }
}
```

## 设计流程

### 1. 需求分析
- 业务目标理解
- 用户需求识别
- 竞品分析
- 约束条件评估

### 2. 信息架构
- 功能梳理
- 内容组织
- 导航结构
- 标签体系

### 3. 交互设计
- 用户流程图
- 任务分析
- 交互模式
- 状态设计

### 4. 视觉设计
- 配色方案
- 字体系统
- 图标系统
- 布局网格
